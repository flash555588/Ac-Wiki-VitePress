# Giscus 评论测试页面

这是一个用于测试 Giscus 评论功能的页面。

## 测试内容

如果你能看到这个页面，说明页面渲染正常。如果你能在页面底部看到评论区域，说明 Giscus 评论功能已经成功集成。

### 功能特性

- ✅ 支持暗黑模式切换
- ✅ 支持中文界面
- ✅ 基于 GitHub Discussions
- ✅ 实时同步评论

### 如何测试

1. 滚动到页面底部
2. 查看是否出现评论区域
3. 尝试切换暗黑模式，看评论区域是否跟随切换
4. 如果有 GitHub 账号，可以尝试发表评论
