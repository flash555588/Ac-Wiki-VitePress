## 本地运行/部署

项目采用 VitePress 构建。

### 安装依赖

```sh
pnpm i
```

### 本地测试

```sh
pnpm run dev
```

### 构建

```sh
pnpm run build
```

### 预览构建结果

```sh
pnpm run preview
```

## 项目结构

- `docs/` - 文档目录
  - `academic-resources/` - 学术资源
  - `campus-life/` - 校园生活
  - `general-skills/` - 通用技能
  - `growth-path/` - 成长路径
  - `community-hub/` - 社区中心
  - `blog/` - 博客文章
