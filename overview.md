# 概览

欢迎来到 Ac-Wiki！这是一个专为大学生打造的学术生活百科。

## 什么是 Ac-Wiki？

Ac-Wiki 是一个综合性的学术生活百科，旨在为大学生提供全面的校园生活指导、学习资源分享和技能培养指南。我们致力于成为大学生群体的"真·青年大学习"平台。

## 我们的使命

- 📚 **学术指导**：提供学习技巧、考试备考、学术工具等资源
- 🏫 **校园生活**：分享校园生活经验，帮助学生快速适应大学生活
- 🛠️ **技能培养**：教授计算机基础、网络安全、生活技能等实用知识
- 🎯 **成长规划**：提供职业规划、实习就业、继续深造等指导
- 🤝 **社区共建**：鼓励学生参与内容贡献，共同建设知识库

## 内容特色

### 校园生活
- 开学第 0 课：帮助新生快速适应大学生活
- 课程安排：了解选课、课表查询等
- 学生服务：校园卡、图书馆、医疗等

### 学术资源
- 学习技巧：高效学习方法、时间管理
- 考试备考：各类考试攻略、复习计划
- 学术工具：推荐实用的学习软件和网站

### 通用技能
- 计算机基础：从零开始的计算机入门
- 网络安全：密码管理、隐私保护
- 生活技能：理财、健康、社交等

### 成长路径
- 职业规划：不同专业的就业方向
- 实习就业：简历制作、面试技巧
- 继续深造：考研、留学等选择

## 如何参与？

我们欢迎所有大学生参与 Ac-Wiki 的建设：

1. **阅读内容**：浏览各个章节，找到你需要的信息
2. **提出建议**：如果你发现内容有误或需要补充，欢迎反馈
3. **贡献内容**：分享你的经验和知识，帮助其他同学
4. **参与讨论**：加入我们的社区，与其他同学交流
